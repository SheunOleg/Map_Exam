оконное приложение wpf

- пользователь может залогиниться\зарегистрироваться

- пользователь может создать маршрут из точек (по умолчанию маршрут не доступен для просмотра другими пользователями)
- маршрут состоит из названия, описания, страны, города, длительности, типа, состояния, списка точек
- точка состоит из координат, названия, описания, изображения
- добавление точки происходит по клику на карте
- пользователь может редактировать маршрут, изменять его состояние (отображается для других пользователей или нет)
- пользователь может редактировать и удалять точки

- пользователь может просмотреть список маршрутов с фильтрами (по стране, городу, длительности, типу)
- просмотр маршрутов реализуется с помощью карт
- при клике на точку маршрута, отображается ее описание

- пользователь может оставить комментарий к маршруту, лайкнуть его (один раз), пожаловаться
- модератор может просмотреть все маршруты, жалобы и отключить отображение маршрута

приветствуется творческий подход и добавление своих фич :)
